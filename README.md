# TCG Arena - Translation

**Translations are managed on [GitLocalize](https://gitlocalize.com/repo/10699).**

* **GitHub account** required.
* **Select your language** to start translating. If your language is missing, please contact me on **Discord** so I can add it.
* Once you are finished (or want to take a break), click **"Create Review Request"** to submit your changes to the app.

Thanks again for your help !
